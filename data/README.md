# 六十四卦资料说明

## 当前文件

- `yijing-64.json`：六十四卦结构化原文资料。
- `../scripts/fetch_yijing_sources.mjs`：从维基文库官方 API 更新原文资料的脚本。

## 每卦已有字段

- `order`：文王卦序。
- `name`、`symbol`：卦名与六十四卦符号。
- `lower`、`upper`：下卦、上卦。
- `guaci`：卦辞。
- `tuan`：彖传。
- `daxiang`：大象传。
- `yaoci`：逐爻爻辞。
- `xiaoxiang`：逐爻小象传。
- `source`：来源名称、网址和获取日期。

## 待补白话字段

每卦预留 `interpretation`，用于项目自己的现代解读：

- `core`：一句话核心判断。
- `situation`：当前处境与发展阶段。
- `career`：工作、事业、合作与决策。
- `relationship`：感情、家庭与人际。
- `wealth`：财务、交易与风险。
- `health`：身心状态提醒，不作医疗诊断。
- `action`：现在适合做什么。
- `warning`：最需要避免什么。

## 建议的页面解卦结构

1. 先给一句明确结论，不让用户先读古文。
2. 解释上下卦如何形成当前局势。
3. 根据用户问题类型显示事业、感情、财运或综合解读。
4. 有变爻时，解释具体爻位及其阶段含义。
5. 最后结合本卦与之卦给出“现状—变化—建议”。
6. 卦辞、彖传、象传和爻辞放在“经典依据”折叠区中。

## 内容边界

- 古籍原文与现代白话解读分开显示，不把现代解释伪装成经文。
- 健康、法律、投资等高风险问题只提供文化参考，不作专业结论。
- 不使用“必然发生”“保证成功”等绝对化措辞。
- 不简单按“吉/凶”二分，要说明时机、条件和行动边界。
